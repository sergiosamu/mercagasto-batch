# Directory Structure

## Root Files
- `main.py` - Main application entry point
- `extract_and_load.py` - Basic product extraction script  
- `extract_and_load_products.py` - Advanced product extraction and database loading

## Scripts
- `scripts/api_tests/` - Scripts for testing Mercadona API endpoints
- `scripts/database/` - Database management and loading scripts
- `scripts/debug/` - Debugging and development tools
- `scripts/legacy/` - Legacy scripts and deprecated code

## Data
- `data/` - Extracted product data and reference files
- `data/extracted_products/` - JSON files with extracted product data

## Core Directories
- `src/` - Main source code
- `tests/` - Test files
- `docs/` - Documentation
- `util/` - Utility modules
- `logs/` - Application logs
- `build/` - Build artifacts and coverage reports

## Configuration
- `.env` - Environment configuration
- `pyproject.toml` - Project configuration
- `uv.lock` / `uv.toml` - Dependency management
